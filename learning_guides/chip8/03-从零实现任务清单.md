## Task 04：实现取指和字段解析

建议先定义辅助函数或宏：

```c
static uint16_t fetch_opcode(const Chip8 *vm);
```

取指：

```c
uint16_t opcode =
    ((uint16_t)vm->memory[vm->PC] << 8) |
    vm->memory[vm->PC + 1];
```

字段：

```c
uint16_t nnn = opcode & 0x0FFF;
uint8_t nn = opcode & 0x00FF;
uint8_t x = (opcode >> 8) & 0x0F;
uint8_t y = (opcode >> 4) & 0x0F;
uint8_t n = opcode & 0x000F;
```

测试数据：

```text
6A3F → X=A，NN=3F
1ABC → NNN=ABC
D125 → X=1，Y=2，N=5
```

完成标准：

- 三组测试全部正确；
- 不越界读取 `PC + 1`；
- 日志能打印 `PC` 与 opcode。

建议提交：

```text
feat: fetch and decode CHIP-8 opcode fields
```

## Task 05：建立单周期执行函数

接口建议：

```c
bool chip8_step(Chip8 *vm);
```

基本流程：

```text
检查 vm 和运行状态
检查 PC 是否越界
取出 opcode
PC 默认加 2
按最高 4 位分派
执行对应指令
错误时停止并返回 false
```

先允许只有少数指令，未知指令打印：

```text
unknown opcode: 0xXXXX at PC=0xXXX
```

完成标准：

- 能连续执行手工构造的指令；
- 未知指令不会静默跳过；
- 错误地址不会导致数组越界。

建议提交：


## Task 08：实现 `DXYN` 绘制

步骤：

1. `start_x = V[X]`；
2. `start_y = V[Y]`；
3. 从 `memory[I]` 开始读取 N 行；
4. 每行读取一个字节；
5. 从最高位到最低位处理 8 个像素；
6. 精灵位为 1 时，与屏幕像素 XOR；
7. 亮像素被擦除时设置 `VF = 1`。

伪代码：

```text
VF = 0
对每一行：
    sprite = memory[I + row]
    对每一位：
        如果 sprite 对应位是 1：
            读取屏幕原像素
            如果原像素是 1：VF = 1
            写入 原像素 XOR 1
```

完成标准：

- 对齐精灵能显示；
- 非字节对齐位置也能显示；
- XOR 再画一次会擦除；
- 碰撞标志正确。

通过测试：

```text
1-chip8-logo.ch8
2-ibm-logo.ch8
```

建议提交：

```text
feat: implement CHIP-8 sprite drawing
```

## Task 09：条件跳过指令

实现：

```text
3XNN：VX == NN 时跳过下一条
4XNN：VX != NN 时跳过下一条
5XY0：VX == VY 时跳过下一条
9XY0：VX != VY 时跳过下一条
```

“跳过下一条”表示：

```c
vm->PC += 2;
```

因为在取指后 `PC` 已经默认增加过 2。

完成标准：

- 条件成立时总共前进 4 字节；
- 条件不成立时只前进 2 字节；
- `5XY0`、`9XY0` 检查最低位必须为 0。

建议提交：

```text
feat: implement CHIP-8 conditional skip opcodes
```

## Task 11：`8XY*` 算术和逻辑指令

实现：

```text
8XY0  VX = VY
8XY1  VX = VX OR VY
8XY2  VX = VX AND VY
8XY3  VX = VX XOR VY
8XY4  VX = VX + VY，VF 为进位
8XY5  VX = VX - VY，VF 为非借位
8XY6  右移，VF 保存移出位
8XY7  VX = VY - VX，VF 为非借位
8XYE  左移，VF 保存移出位
```

重要规则：

- 在修改 `VX` 前保存原始操作数；
- 尤其注意 `X == F` 或 `Y == F`；
- Shift 指令存在不同历史行为，第一版先选定一种并记录。

完成标准：

- 进位、借位、移位标志正确；
- 8 位溢出自然截断；
- 通过 Flags 测试。

建议提交：

```text
feat: implement CHIP-8 arithmetic and logic opcodes
```

## Task 13：实现 `FX**` 指令

推荐顺序：

```text
FX07：VX = delay_timer
FX15：delay_timer = VX
FX18：sound_timer = VX
FX1E：I += VX
FX29：I 指向 VX 对应字体
FX33：BCD
FX55：V0～VX 写入内存
FX65：从内存读取 V0～VX
FX0A：等待按键
```

BCD 示例：

```text
VX = 231
memory[I]     = 2
memory[I + 1] = 3
memory[I + 2] = 1
```

完成标准：

- 内存操作有边界检查；
- `FX0A` 真正暂停指令推进；
- 明确记录 `FX55/FX65` 后是否改变 `I`。

建议提交：
